# Argus Community Code of Conduct

Argus follows the CNCF Community Code of Conduct:

**https://github.com/cncf/foundation/blob/main/code-of-conduct.md**

## Reporting

If you experience or witness unacceptable behavior, or have any other concerns,
please report it by contacting the project maintainers at:

**argus.telemetry@gmail.com**

All reports will be handled with discretion. You may also report issues directly
to the CNCF Code of Conduct Committee at conduct@cncf.io.

## Scope

This Code of Conduct applies within all project spaces — GitHub repositories,
issue trackers, community calls, and any other forums managed by Argus maintainers.
It also applies when an individual is representing the project in public spaces.
